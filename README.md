# 📊 Merchant Churn Analysis

## 📌 Sobre o Projeto

Este projeto tem como objetivo analisar os fatores relacionados ao churn (rotatividade) de comerciantes por meio de técnicas de Análise Exploratória de Dados (EDA) e, posteriormente, desenvolver um modelo de Machine Learning capaz de prever a probabilidade de cancelamento.

Além da modelagem preditiva, o projeto contempla a construção de um dashboard interativo para apresentação dos principais indicadores e insights obtidos durante a análise.

---

## 🎯 Objetivos

* Explorar e compreender os dados.
* Identificar padrões e comportamentos dos comerciantes.
* Investigar fatores associados ao churn.
* Desenvolver um modelo de classificação para previsão de churn.
* Disponibilizar os resultados em um dashboard interativo.

---

## 🚀 Etapas do Projeto

* ✔️ Coleta e carregamento dos dados.
* ✔️ Exploração inicial da base de dados.
* ✔️ Análise Exploratória de Dados (EDA).
* 🔄 Engenharia de atributos *(em andamento)*.
* ⏳ Preparação dos dados para Machine Learning.
* ⏳ Treinamento e avaliação dos modelos.
* ⏳ Construção do dashboard interativo.
* ⏳ Documentação dos resultados e conclusões.
